# Day 03 – Linux Commands Practice

## Task

## Today’s goal is to build your Linux command confidence.

## Process management
- ps --> Show running processes
- top --> Live view on system usage (CPU,memory)
- kill --> Stop a process
- htop --> Better version of top command (if installed)
- pgrep <name> --> find process id by name 

## File system
- pwd --> Show current directory
- ls --> List files and folders
- cd --> Change directory
- mkdir --> Make a directory
- touch --> Create a empty file
- cp --> Copy files / folders
- mv --> Move or renames files
- rm --> Delete files / folders
- clear --> Clear the terminal
- df -h --> Check disk usage
- du -sh --> Check folder usage

## Networking troubleshooting
- ping -->  Check if host is reachable
- dig --> Advanced DNS info
- ip addr --> Interface and routing config
- curl,wget --> Test HTTP/HTTPS connections
